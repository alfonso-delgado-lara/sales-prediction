# Predictive Modeling for Sales Prediction

## Overview
This project focuses on building and evaluating predictive models using real-world data from an advertising company. It follows a structured data science workflow, from exploratory data analysis to model selection, with a strong emphasis on analytical reasoning and methodological rigor.

Developed as part of a Master's program in Big Data at Universidad Carlos III de Madrid.

---

## Objective
The goal of this project is to analyze the dataset, extract meaningful insights, and develop predictive models capable of explaining and forecasting the target variable.

---

## Approach

### 1. Data Preprocessing
- Data cleaning and handling of missing values
- Transformation of variables
- Preparation of data for modeling

### 2. Exploratory Data Analysis (EDA)
- In-depth analysis of variable distributions
- Identification of relationships between variables
- Detection of patterns and potential predictors

### 3. Feature Engineering
- Creation and transformation of variables
- Selection of relevant features for modeling

### 4. Modeling
Several models were implemented and compared, including:
- Linear models with strong focus on variable selection.
- Tree-based methods (CART,Random Forest, Gradient Boosting)
- K-Nearest Neighbours

### 5. Model Evaluation
- Performance comparison using Mean Absolute Error.
- Selection of the most suitable model
- Analysis of model behavior
- Control of overfitting for future performance.

---

## Results
The project highlights the importance of combining strong exploratory analysis with appropriate modeling techniques to achieve reliable and interpretable results.
